# 🍉 Watermelon BLAST Tool

一个基于 **Watermelon (97103) v2.5 Genome** 的在线 BLAST 序列比对工具。  
该项目支持 DNA 短序列的本地比对查询，提供中文界面、阈值筛选、命中序列过滤、最佳命中显示、覆盖率计算、结果下载，以及单匹配 / 多匹配自动判定等功能。

---

## 项目简介

本项目是一个部署在 Linux 云服务器上的 Web BLAST 工具，面向西瓜基因组（97103 v2.5）进行 nucleotide-to-nucleotide（n 对 n）序列比对。

相比基础版 BLAST 页面，本项目增加了更适合展示和使用的功能，包括：

- 中文专业版界面
- 支持纯序列输入与 FASTA 输入
- 支持多条 FASTA 序列批量查询
- 支持命中序列筛选（如 Cla97Chr03）
- 支持“仅显示最佳命中（Top hit）”
- 支持按 E-value / Bit score / 相似度 / 覆盖率排序
- 支持“有效匹配 / 低质量命中 / 非匹配”自动判定
- 支持覆盖率计算
- 支持结果下载为 CSV / TXT
- 支持简单位置图可视化
- 支持单匹配 / 多匹配自动判定

---

## 核心匹配逻辑

本项目的检索流程不是“只看某一条染色体”，而是：

1. **先对整个西瓜基因组（所有染色体）进行 BLAST 搜索**
2. **再通过阈值规则自动过滤低质量命中**
3. **最后根据过滤后的结果数量自动判定为：**
   - 不匹配
   - 单匹配
   - 多匹配

### 默认过滤规则

系统可根据用户设置进行过滤，核心指标包括：

- 最小一致碱基数（alignment length）
- 最小相似度（identity %）
- 最大 E-value

例如：

- 最小一致碱基数：20
- 最小相似度：100%
- 最大 E-value：0.05

在这种情况下，只有同时满足以下条件的结果才会被判定为有效匹配：

- 比对长度 ≥ 20
- 相似度 ≥ 100%
- E-value ≤ 0.05

### 单匹配 / 多匹配判定

过滤后：

- 剩余 0 条结果 → **不匹配**
- 剩余 1 条结果 → **单匹配**
- 剩余 2 条及以上结果 → **多匹配**

该逻辑更接近真实 BLAST 工具对短序列匹配结果的使用方式，也避免了低质量局部命中造成的误判。

3. 阈值判定

用户可自定义：

最小一致碱基数
最小相似度（%）
最大 E-value

系统会据此自动过滤命中结果，并更新匹配判定。

4. 命中序列过滤

支持按特定染色体筛选结果，例如：

Cla97Chr03
Cla97Chr11
或全部显示
5. Top Hit 模式

支持：

显示全部结果
仅显示最佳命中（Top 1）
每条 Query 仅显示最佳命中
6. 结果排序

支持以下排序方式：

E-value 从小到大
Bit score 从大到小
相似度从高到低
覆盖率从高到低

支持下载：

CSV
TXT

结果表格中展示：

命中序列名
